# Auth Server

#### GET `/api/register/init`
```json
{
	"type": "message",
	"log": {
		"message": "Started user registration progress",
		"log_level": 0
	},
	"data": {
		"uuid": "<uuid>"
	}
}
```



#### POST `/api/register/complete`
`REQ`
```json
{
	"uuid": "",
	"public_key": "",
	"private_key_hash": "",
	"username": ""
}
```
`RES`
```json
{
	"type": "message",
	"log": {
		"message": "Created User: <uuid>",
		"log_level": 0
	},
	"data": {}
}
```
